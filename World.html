HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Es un lenguaje de marcado que define la estructura y el contenido de una página web mediante una serie de elementos y etiquetas. HTML permite a los desarrolladores web definir distintos tipos de elementos en una página, como encabezados, párrafos, enlaces, imágenes, listas, formularios, entre otros. Cada elemento HTML se define mediante etiquetas que rodean el contenido y proporcionan información sobre su función y estructura.
